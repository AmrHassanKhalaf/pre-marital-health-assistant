# 💍 رفيقة — Pre-Marital Health Assistant

مساعدة صحية تعليمية وقائية مخصصة لدعم الأشخاص المقبلين على الزواج في الاستعداد الصحي قبل الزواج.

## 🌟 المميزات

- **توعية صحية قبل الزواج** — شرح مبسط وعملي
- **نظام RAG هجين** — يعتمد على قاعدة معرفة + توجيه عام آمن
- **تصعيد أمان ذكي** — عند طلب التشخيص أو القلق العالي يوجّه لمختص
- **واجهة عربية** — تصميم RTL مناسب ومريح
- **لوحة تحكم** — متابعة المحادثات والطوارئ

## 📋 النطاق

- فحوصات ما قبل الزواج (Blood group/Rh, CBC, عدوى، توعية بالفحوصات الجينية)
- الصحة الإنجابية والهرمونية الأساسية
- التوعية بالخصوبة والعمر ونمط الحياة
- التغذية والنشاط البدني وإدارة الوزن
- النوم والتوتر والتدخين
- الجاهزية النفسية والعاطفية قبل الزواج
- متى تراجع الطبيب وأي تخصص أنسب

## 📁 هيكل المشروع

```
chatbot/
├── app.py              # تطبيق Gradio (للنشر على HF Spaces)
├── store_index.py      # بناء قاعدة البيانات في Pinecone
├── requirements.txt    # المكتبات المطلوبة
├── .env.example        # متغيرات البيئة (نموذج)
└── src/
    ├── __init__.py
    ├── prompt.py        # System Prompt الخاص برفيقة
    ├── helper.py        # دوال مساعدة (تحميل، تقسيم، embeddings)
    ├── emergency.py     # نظام تصعيد الأمان
    └── admin_store.py   # تخزين بيانات لوحة التحكم
```

## 🚀 التشغيل المحلي

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. إعداد متغيرات البيئة
```bash
cp .env.example .env
# قم بتعبئة API keys في ملف .env
```

### 3. بناء قاعدة البيانات (أول مرة فقط)
```bash
python store_index.py
```

### 4. تشغيل الشات بوت
```bash
python app.py
```

## 🚨 قواعد الأمان

عند الحالات التالية يتم التصعيد فورًا:

- طلب تشخيص طبي مباشر
- طلب تفسير سريري حاسم لنتائج التحاليل
- قلق شديد أو ذعر مرتبط بالفحوصات
- مشاركة نتيجة مقلقة بشكل يتطلب تقييم طبي
- أي مؤشرات خطر مباشر على النفس

الرد يكون: طمأنة + توضيح الحدود + توجيه لطبيب مختص.

## 📝 ملاحظة

رفيقة مساعدة إرشادية فقط ولا تُغني عن استشارة الطبيب المختص.
